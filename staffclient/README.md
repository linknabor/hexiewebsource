############################################################################################
命名规范；
vue文件: 除了index.vue，都PascalCase风格。
js文件: 除了util包下的js，都PascalCase风格，如果是接口性质的，后缀跟Api
页面上写组件，以kebab-case为主要风格，比如<user-info></user-info>

代码风格
遵从vue风格，每一行最后不要使用逗号。请自行关闭eslint对逗号的校验。
